# classnova
student management system for college
Student Management System

A simple and intuitive Student Management System (SMS) web application that allows administrators to manage student records efficiently. Built with Node.js, Express, MySQL, and a responsive frontend using HTML/CSS/JavaScript.

Features
Add, edit, and delete student records
View all students in a table format
Search students by name, roll number, or course
Responsive design for desktop and mobile
Professional and user-friendly interface
Login and authentication system for secure access

Technologies Used
Backend: Node.js, Express.js
Database: MySQL
Frontend: HTML5, CSS3, JavaScript
Other: CORS, Path module
